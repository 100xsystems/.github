## What is 100xSystems?

A learning platform where you build real software systems from scratch. Not tutorials, not courses — **you write the code, we validate it.**

Each system has:
- 📚 **Curriculum** — Lessons that teach you the architecture
- 🧪 **Behavioral tests** — Real test suites that verify your implementation
- ✅ **Validation** — The CLI tests your code against the curriculum
- 📤 **Submission** — Package your work and submit for review

## Available Systems

| System | Description | Tracks |
|--------|-------------|--------|
| [Claude Code](https://github.com/100xsystems/claude-code) | Build an AI-powered coding agent | TypeScript, Java |
| [Microservices](https://github.com/100xsystems/microservices) | Master distributed systems architecture | Spring Boot, NestJS |

## Quick Start

```bash
# Install the CLI
npm install -g @100xsystems/cli

# Pick a system and start building
100xsystems list
100xsystems init claude-code
cd claude-code-implementation
100xsystems validate
```
